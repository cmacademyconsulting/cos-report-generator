## The COS™ Methodology

COS™ (Compliance · Oversight · Sustainability) is an open-access 
construction governance framework published on Zenodo.

Scores are never adjusted to satisfy an authority. A project 
scoring 41% on Oversight is reported as 41%. That honesty is 
the methodology's foundation.
